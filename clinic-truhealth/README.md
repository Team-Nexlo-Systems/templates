# TruHealth Clinic — Next.js Website

Built with Next.js 14, Tailwind CSS, App Router. No external UI libraries.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Clinic Photo

Already included at `/public/clinic.jpg` (signboard photo).

## Deploy to Vercel

1. Push to GitHub
2. vercel.com → New Project → Import → Deploy

## Sections

- Navbar (sticky, mobile responsive)
- Hero (clinic photo, WhatsApp + Call CTAs, stats)
- Services (9 cards from signboard — hover animation)
- Our Team (3 doctors/nutritionist from signboard)
- Testimonials (6 real Google reviews)
- Timings & Location (table + Google Maps embed)
- Contact (form with success state)
- Footer (Nexlo Systems credit)
